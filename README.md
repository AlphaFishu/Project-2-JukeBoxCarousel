# Project-2-JukeBoxCarousel

A premium 3D Vintage Jukebox Carousel web application featuring an interactive 3D cylinder rotating carousel of iconic music albums.

## GitHub Repository
[https://github.com/AlphaFishu/Project-2-JukeBoxCarousel](https://github.com/AlphaFishu/Project-2-JukeBoxCarousel)

## Features
- **3D Album Carousel**: Smooth interactive 3D rotation of album covers using standard polygon physics.
- **Spotify/iTunes Cover Art**: Displays 600x600 high-resolution album covers.
- **Dynamic Backdrop/Lighting**: Background dynamically shifts using color theory based on the dominant colors of the active album cover.
- **Virtual Scroll & Gesture Control**: Support for mouse wheel scrolling (desktop) and touch swipes (mobile) with physics-based linear interpolation (LERP) for smooth movement.
- **Responsive Layout**: Adapts gracefully across desktop and mobile devices.

## Running Locally
To run the server locally:
```bash
python3 -m http.server 8000
```
Then open [http://localhost:8000](http://localhost:8000) in your web browser.
